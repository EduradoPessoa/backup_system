Início Rápido

Faça seu primeiro backup em 3 passos simples:

1
Selecione Pastas

Clique em "Adicionar Pasta" e escolha as pastas que deseja fazer backup

2
Configure Destino

Escolha onde salvar o backup e dê um título descritivo

3
Iniciar Backup

Clique em "Iniciar Backup" e acompanhe o progresso

Como Criar um Backup

1. Seleção de Pastas

  • Adicionar Pasta: Clique no botão e navegue até a pasta desejada
  • Múltiplas Pastas: Você pode adicionar quantas pastas quiser
  • Remover Pasta: Clique no "X" ao lado da pasta para removê-la

2. Destino do Backup

  • Escolha uma pasta diferente das que está fazendo backup
  • Certifique-se de ter espaço suficiente no destino
  • Recomendamos usar um disco externo ou pasta na nuvem

3. Título do Backup

  • Dê um nome descritivo como "Documentos_Janeiro_2025"
  • O sistema adiciona data e hora automaticamente
  • Evite caracteres especiais como / \ : * ? " < > |

4. Formato de Compressão

ZIP

Rápido
Melhor para backups frequentes

TAR.GZ

Balanceado
Boa compressão e velocidade

7Z

Máxima Compressão
Menor tamanho de arquivo

Recursos Avançados

Backup Incremental

Faz backup apenas de arquivos novos ou modificados desde o último backup, economizando tempo e espaço.

Como usar: Marque a opção "Backup Incremental" antes de iniciar

Incluir Subpastas

Inclui automaticamente todas as subpastas e arquivos dentro das pastas selecionadas.

Padrão: Sempre habilitado para backup completo

Progresso em Tempo Real

Acompanhe o progresso do backup com duas fases:

  • Análise (0-80%): Calculando tamanho dos arquivos
  • Backup (80-100%): Comprimindo e salvando
Tratamento de Erros

Sistema robusto que lida com arquivos bloqueados ou corrompidos:

  • Pula arquivos com problema automaticamente
  • Gera relatório detalhado de erros
  • Permite continuar o backup mesmo com problemas

Como Restaurar um Backup

1. Localizar Backup

  • Use a aba "Catálogo" para ver todos os backups realizados
  • Ou use "Restaurar Backup" para selecionar um arquivo específico
  • Visualize detalhes como data, tamanho e número de arquivos

2. Inspecionar Conteúdo

  • Visualize o conteúdo do backup antes de restaurar
  • Navegue pela estrutura de pastas
  • Selecione arquivos específicos se necessário

3. Extrair Arquivos

  • Escolha a pasta de destino para restauração
  • Confirme que não vai sobrescrever arquivos importantes
  • Acompanhe o progresso da extração

Solução de Problemas

Soluções:
  • Use formato ZIP para backups mais rápidos
  • Evite fazer backup durante uso intenso do disco
  • Feche outros programas pesados
  • Use backup incremental para backups subsequentes

O sistema lida automaticamente:
  • Arquivos em uso são pulados automaticamente
  • Backup continua com os arquivos disponíveis
  • Relatório mostra quais arquivos foram pulados
  • Feche programas que possam estar usando os arquivos

Soluções:
  • Use formato 7Z para máxima compressão
  • Faça backup de pastas menores separadamente
  • Limpe arquivos desnecessários antes do backup
  • Use um disco externo com mais espaço

Dicas Importantes

Segurança
  • Sempre teste a restauração de seus backups
  • Mantenha backups em locais diferentes
  • Use a regra 3-2-1: 3 cópias, 2 mídias, 1 externa
Frequência
  • Documentos importantes: backup diário
  • Projetos em andamento: backup semanal
  • Arquivos pessoais: backup mensal
